# FireNetStats Mockups

Este directorio contiene activos visuales utilizados para ilustrar el diseño
propuesto de la aplicación **FireNetStats**.

## mockup_firenetstats.png

Un mockup de alta fidelidad simplificado que muestra:

- La pantalla principal con panel de estado, historial, botones de acción y
  estadísticas adicionales.
- El widget flotante draggable con las velocidades de descarga y carga.
- Notas que describen interacciones clave.

Este recurso sirve como referencia visual para el diseño descrito en las
iteraciones previas.
